            Cryptocurrency is a digital or virtual currency 
            that uses cryptography for security.


Cryptocurrencies work using blockchain technology, which is a distributed ledger system that records all transactions across a network of computers. Transactions are verified and added to the blockchain through a process called mining.Each cryptocurrency has its own blockchain, and transactions are secure and transparent. Examples of cryptocurrencies include Bitcoin, Ethereum, Ripple, and Litecoin.
